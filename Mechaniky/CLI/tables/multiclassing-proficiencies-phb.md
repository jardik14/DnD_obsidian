# Multiclassing Proficiencies
*Source: Player's Handbook p. 164. Available in the <span title='Systems Reference Document (5.1)'>SRD</span>* 

**Multiclassing Proficiencies**

| Class | Proficiencies Gained |
|-------|----------------------|
| [Barbarian](/Mechaniky/CLI/classes/barbarian.md) | Shields, simple weapons, martial weapons |
| [Bard](/Mechaniky/CLI/classes/bard.md) | Light armor, one skill of your choice, one musical instrument of your choice |
| [Cleric](/Mechaniky/CLI/classes/cleric.md) | Light armor, medium armor, shields |
| [Druid](/Mechaniky/CLI/classes/druid.md) | Light armor, medium armor, shields (druids will not wear armor or use shields made of metal) |
| [Fighter](/Mechaniky/CLI/classes/fighter.md) | Light armor, medium armor, shields, simple weapons, martial weapons |
| [Monk](/Mechaniky/CLI/classes/monk.md) | Simple weapons, shortswords |
| [Paladin](/Mechaniky/CLI/classes/paladin.md) | Light armor, medium armor, shields, simple weapons, martial weapons |
| [Ranger](/Mechaniky/CLI/classes/ranger.md) | Light armor, medium armor, shields, simple weapons, martial weapons, one skill from the class's skill list |
| [Rogue](/Mechaniky/CLI/classes/rogue.md) | Light armor, one skill from the class's skill list, [thieves' tools](/Mechaniky/CLI/items/thieves-tools.md) |
| [Sorcerer](/Mechaniky/CLI/classes/sorcerer.md) | — |
| [Warlock](/Mechaniky/CLI/classes/warlock.md) | Light armor, simple weapons |
| [Wizard](/Mechaniky/CLI/classes/wizard.md) | — |
