# Book
*Adventuring gear*  


- **Cost**: 25 gp
- **Weight**: 5.0 lbs.

A book might contain poetry, historical accounts, information pertaining to a particular field of lore, diagrams and notes on gnomish contraptions, or just about anything else that can be represented using text or pictures. A book of spells is a [spellbook](/Mechaniky/CLI/items/spellbook.md).

*Source: Player's Handbook p. 151. Available in the <span title='Systems Reference Document (5.1)'>SRD</span> and the Basic Rules (2014)*
