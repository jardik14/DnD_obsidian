# Healer's Kit
*Adventuring gear*  
![](/Mechaniky/CLI/items/img/healers-kit.webp#right)

- **Cost**: 5 gp
- **Weight**: 3.0 lbs.

This kit is a leather pouch containing bandages, salves, and splints. The kit has ten uses. As an action, you can expend one use of the kit to stabilize a creature that has 0 hit points, without needing to make a Wisdom ([Medicine](/Mechaniky/CLI/Rules/skills.md#Medicine)) check.

*Source: Player's Handbook p. 151. Available in the <span title='Systems Reference Document (5.1)'>SRD</span> and the Basic Rules (2014)*
